You're on Desktop!

You can use your mouse, but these demo's are much cooler on your phone or tablet!

Chat Heads

A "chat heads" chat overlay, much like facebook's. This demo uses the interaction, spring, and attachSpring methods.

There is a main chat head, which you drag around, and then each additional head is attached to the next, kindof like a linked list!

pulldownMenu

A pull down menu, with a feel like the ios 7 notifications pulldown-menu. Accelerates down and bounces as if pulled by gravity, and springs closed.

GithubTry ItTry It

Inertia Scroll

Inertia scrolling with inertia, deceleration, overscroll and bounce, in about 30 lines.

Chat bubble and header designed with Framework7, an amazing library for building web apps with ios 7's look.

GithubTry ItTry It

Note

This example shows how concise Impulse is, but mobile web browsers already do scrolling well by default. This may not be something you want to do in production.

Oridomi Cover

This example uses the awesome oridomi library as a custom renderer.

Normally oridomi elements don't animate open and closed on swipe. We used impulse to add the ability to open and close the "cover" with just a few lines in Impulse.

Bouncy Scrolling

BETA

demo is using an experimental optimization branch of impulse

Taking the previous example to the extreme, we added bouncy scrolling, which replicates the iOS 7 Messages App bouncy scroll. Due to the number of compositor RenderLayers (each message) and the number of springs that need to be simulated, this example may be slower on older devices.
